# Get Started

Digi Design Language is a reusable component library that helps developer build UIs faster. The goal is to make building durable UIs more productive and satisfying.

<div className="subheading">Installation</div>

DDL components are written in React, and its stories are written in Component Story Format. It requires Storybook version 5.2-beta and up.

Add DDL to your project.
<code>npm install --save @storybook/design-system</code>
